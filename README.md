# Pharmacare

Pharmacare is an online pharmacy platform built using Django and React. The website provides a seamless way for users to buy medicines, book doctor appointments, and purchase healthcare packages.

## Features

- **Buy Medicines:** Browse and purchase medicines from a variety of categories.
- **Doctor Appointments:** Book appointments with doctors based on their specializations.
- **Healthcare Packages:** Purchase comprehensive healthcare packages tailored to your needs.
- **User Profile:** View and manage your details, purchase history, and upcoming appointments.
- **Secure Payments:** Integrated with Stripe for secure and smooth payment processing.
- **Search and Filter:** Advanced search and filtering options for easy navigation through products and services.
- **Pagination:** User-friendly pagination on product pages for easy browsing.

## Tech Stack

- **Frontend:** React, Tailwind CSS
- **Backend:** Django, Django REST Framework
- **Database:** PostgreSQL
- **Payments:** Stripe
